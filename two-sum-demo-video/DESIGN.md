# Two Sum Demo Video Design

## Style Prompt

Clean technical product explainer for AI Interview Coach Agent. The visual language should feel like a polished Java backend demo: precise, readable, calm, and interview-ready. Use the application's existing blue-on-light surface palette, code-editor dark panels, and structured diagnostic cards. Motion should feel like a guided workflow rather than a flashy ad.

## Colors

- Background: `#faf8ff`
- Surface: `#ffffff`
- Surface raised: `#eaedff`
- Primary: `#005a90`
- Primary bright: `#0073b7`
- Text: `#131b2e`
- Muted text: `#404750`
- Error: `#ba1a1a`
- Success: `#15803d`
- Warning: `#b7791f`
- Code panel: `#0d1117`
- Code text: `#e6edf3`

## Typography

- UI and headings: `Microsoft YaHei`, `IBM Plex Sans`, sans-serif
- Code and data: `JetBrains Mono`, monospace
- Use large, readable Chinese headings and compact code blocks.

## What NOT to Do

- Do not use neon cyberpunk colors.
- Do not use a generic dark purple/blue gradient background.
- Do not show a right-side "分层提示" tab; the current product separates left preset hints from right AI diagnosis.
- Do not imply AI generates the preset problem hints on click.
- Do not show full accepted answer as AI output; the fixed code is a deliberate demo artifact.
